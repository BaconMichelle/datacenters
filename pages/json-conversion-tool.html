JSON Conversion Tool

Introduction

The JSON conversion tool has been developed to support agences in submitting their DCOI Strategic Plans which are to be published in a machine-readable JSON format. This can be found at ‘agency’.gov/digitalstrategy under a section entitled, “Data Center Optimization Initiative Strategic Plans.”

Thank you for taking the time to enter the information below.

Guidance

Please enter the relevant data into each fields below. Data can be entered manually or automatically uploaded by clicking the ‘Load my own data’ button on the screen.

Once the ‘Load my own data’ button is clicked, a dialogue box will appear prompting you to enter your data in a JSON format. Once entered, click the ‘Load data’ button at the bottom of the screen.

Please Note: Some fields will only accept numerical data. In the event that the data does not match the parameters for a specific field, an error statement will appear describing the type of data to be entered to correct the error.

After entering the data, click the submit button at the very bottom of the screen. The tool will convert the entered data into the JSON format which will display in a field at the bottom of the screen. Copy the JSON format output into a text file and save for formal submission to OMB.

Please Note: Clicking submit does not indicate that the output has been submitted to OMB. The output must still be submitted to OMB after the JSON schema is created by this tool. For additional guidance, please visit management.cio.gov/schema>.
