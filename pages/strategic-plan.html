Annual Strategic Plan JSON Conversion Tool

Data Center Optimization Initiative (DCOI)

Introduction

In preparation for the forthcoming OMB DCOI Strategic Plan, the following conversion tool will support Agencies’ ability to prepare JSON file submissions. Strategic Plans then must be published in a machine-readable JSON format at ‘agency’.gov/digitalstrategy under a section entitled, “Data Center Optimization Initiative Strategic Plans.”

Thank you for taking the time to enter the information below.

Guidance

Please enter the relevant data into each box. You can either enter the data manually or upload the data automatically.

To upload the data automatically, please click the ‘Load my own data’ button below, after which a new dialogue box will appear. In this box, please enter your data in a JSON format and then click the ‘Load data’ button at the bottom. Note that some boxes will only accept numerical data. In the event that the data does not match the parameters for a specific box, an error statement will appear below describing what sort of data is required to fix the error.

After filling in the data, either manually or by an upload, click the submit button at the very bottom. The tool will produce the entered data in a JSON format in a box at the very end. Please copy this output into a text file and save it in a JSON format for formal submission to OMB. Please note that clicking submit does not mean that the output is actually submitted to OMB. The output must still be submitted to OMB after the JSON schema is created by this tool. For additional guidance, please visit management.cio.gov/schema.
